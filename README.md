# Журнал Искателя приключений — пакет для сборки APK

Это готовое PWA-приложение. Из него можно бесплатно собрать APK через PWABuilder.

## Что внутри
- `index.html` — приложение (шрифты встроены локально, работает офлайн)
- `manifest.json` — описание приложения (имя, иконки, цвета)
- `sw.js` — service worker (кэширует всё для работы без интернета)
- `fonts/` — шрифты Inter и Cinzel
- `icons/` — иконки приложения

## Как получить APK (примерно 10 минут)

1. Выложи содержимое папки в интернет. Проще всего — **Netlify Drop**:
   зайди на https://app.netlify.com/drop и перетащи туда **эту папку**
   (именно папку целиком). Получишь адрес вида `имя-сайта.netlify.app`.
   Регистрация не нужна.

2. Открой https://www.pwabuilder.com , вставь полученный адрес и нажми Start.
   Сервис проверит манифест и service worker — всё должно быть зелёным.

3. Нажми **Package for stores → Android**. Можно оставить настройки по
   умолчанию. Скачается zip, внутри — файл `.apk` (для установки на телефон)
   и `.aab` (для загрузки в Google Play, если понадобится).

4. Перекинь `.apk` на телефон и установи. Возможно, потребуется разрешить
   «установку из неизвестных источников».

## Обновление приложения
Если поменяешь `index.html`, открой `sw.js` и увеличь версию кэша
(`dnd-journal-v1` → `dnd-journal-v2`), иначе у пользователей останется
старая версия из кэша. Затем снова выложи папку и пересобери APK.
