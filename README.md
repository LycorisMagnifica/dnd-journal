# Журнал Искателя приключений — пакет для сборки APK

Это готовое PWA-приложение. Из него можно бесплатно собрать APK через PWABuilder.

## Что внутри
- `index.html` — приложение (шрифты встроены локально, работает офлайн)
- `manifest.json` — описание приложения (имя, иконки, цвета)
- `sw.js` — service worker (кэширует всё для работы без интернета)
- `fonts/` — шрифты Inter и Cinzel
- `icons/` — иконки приложения
